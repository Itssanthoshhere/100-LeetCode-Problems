# Problem 25 - Majority Element II

**Difficulty:** Medium  
**Platform:** LeetCode

## Problem Statement

Given an integer array `nums` of size n, find all elements that appear more than ⌊ n/3 ⌋ times.

## Examples

**Example 1:**  
```
Input: nums = [3,2,3]
Output: [3]
```

**Example 2:**  
```
Input: nums = [1]
Output: [1]
```

**Example 3:**  
```
Input: nums = [1,2]
Output: [1,2]
```
